RAFT consensus algorithm implementation

Usage
-----
.. code-block:: bash

        raftnode --ip MY_IP:MY_PORT --peers PEER1:PORT1,PEER2:PORT2,...,PEERn:PORTn 

For more detailed CLI

.. code-block:: bash

        raftnode --help
